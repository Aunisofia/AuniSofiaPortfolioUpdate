Vehicle Booking System

Vehicle Booking System for Hasta Travel & Tours

A collaborative web development project created during our Work-Based Learning program - our first experience building a real-world, fully functional website.

Project Overview:

In partnership with Hasta Travel & Tours, my team and I developed a comprehensive vehicle booking system. This project marked our first end-to-end web development experience, from initial concept to deployment.

Project Timeline:

⌛2025 (System Analysis & Design): Designed the interface and gathered initial user requirements
⌛2026 (Third Semester): Developed the database architecture, built system components, and conducted multiple stakeholder meetings to refine requirements. Despite some requirement changes, we successfully adapted and implemented all necessary adjustments.

Technical Implementation:

🛠️Built with Laravel as requested by stakeholders
🛠️Database design and management using MySQL through XAMPP
🛠️Version control with GitHub Desktop
🛠️Development in Visual Studio
🛠️Deployed on InfinityFree hosting

Key Features:

For Customers:

👥Car booking functionality
👥Vehicle inspection process
👥Booking history and details view
👥Profile information management

For Staff:

👩🏻‍💻Payment approval/rejection system
👩🏻‍💻Payment details viewing
👩🏻‍💻Commission management for after-hours services
👩🏻‍💻Access to customer inspection records

For Admin:

💠Comprehensive booking oversight
💠User management (customers, staff, admins) including blacklist functionality
💠Commission approval system
💠Vehicle inventory management
💠Inspection details monitoring
💠Report generation (revenue, total bookings, top colleges, blacklisted customers)

This project was a significant learning journey for our team. As first-time developers building a production-ready website, we encountered various challenges but persevered through each obstacle. During Industry Day, we received valuable feedback from both our lecturer and stakeholders where the insights we wholeheartedly embraced and will apply to future projects.

This project not only enhanced our technical skills but also taught us the importance of adaptability, stakeholder communication, and continuous improvement in real-world development scenarios.
