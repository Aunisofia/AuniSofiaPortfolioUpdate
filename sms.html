Student Management System

Project Overview:

A project developed outside class for the System Development Technology (SDT) Test. I developed a System Management System for the SDT subject to test my understanding and push my development capabilities further.

Project Timeline:

⌛January 2026 (SDT): Designed and built the system from scratch

Technical Implementation:

🛠️Built using the traditional method
🛠️Database design and management using MySQL through XAMPP
🛠️Development in Visual Studio
🛠️Deployed on InfinityFree hosting

Key Features:

For Students:

👩🏻‍🏫View course details
👩🏻‍🏫Register new course for new semester
👩🏻‍🏫Modify the course registration
👩🏻‍🏫View history registered course from previous semester
👩🏻‍🏫Profile update

For Lectures:

✨Manage course (add new course, modify course)
✨Register new student and lecturer
✨Can communicate through the student information
✨Manage student and staff details

For Admin:

💠Comprehensive booking oversight
💠User management (customers, staff, admins) including blacklist functionality
💠Commission approval system
💠Vehicle inventory management
💠Inspection details monitoring
💠Report generation (revenue, total bookings, top colleges, blacklisted customers)

This project gave me opportunity to design the entire surface and build the database from scratch. This was my first solo web development project and I'm excited to continue improving my skills in future builds.
